# AI as Your TA

A growing collection of AI-based tools and resources aimed at helping educators work more efficiently. The goal is to track, evaluate, and document AI tools that support teaching — from lesson planning and slide generation to grading and student interaction.

## Repository Structure

```
ai_as_you_ta/
└── resources/          # Curated AI tool lists and the prompts used to generate them
    ├── instructions.txt    # Claude Code prompt used to research and build the resource list
    └── resource_list.md    # Categorized list of AI teaching tools with URLs and descriptions
```

## What's Here

### `resources/resource_list.md`
A curated, categorized markdown table of AI tools useful for K-12 and higher-ed teachers. Categories include:

- **AI Slide & Presentation Generation** — tools like Gamma, SlidesAI, and Plus AI
- **AI Lesson Plan Tools** — tools like Khanmigo, Eduaide, and Diffit
- **AI Grading & Assessment** — tools like Gradescope, Turnitin, and Formative
- **AI Quiz & Worksheet Generators** — tools like ClassPoint AI and Twee
- **All-in-One AI Platforms** — tools like MagicSchool AI and Brisk Teaching

### `resources/instructions.txt`
Documents the prompt(s) used with the Claude Code CLI (`claude instructions.txt`) to research and generate the resource list, including notes on how the prompt was refined after the initial run.

## How This Was Built

The resource list was generated by feeding `instructions.txt` directly to the Claude Code CLI, which searched the web and produced a structured markdown output. The prompt was iterated on to filter out blog/review pages and surface only actual tools and products.
